🌐 公网访问解决方案

✅ 方案一：使用云服务（最简单）

将问卷系统部署到免费的云平台，获得公网访问地址：

推荐平台：

  1. GitHub Pages - 免费静态网站托管
    • 上传问卷HTML文件到GitHub仓库
    • 开启GitHub Pages功能
    • 获得：https://[用户名].github.io/[仓库名]/
  2. Vercel - 免费部署，支持HTTPS
    • 拖拽上传文件夹
    • 自动生成域名
    • 全球CDN加速
  3. Netlify - 类似Vercel，也很简单

🔧 方案二：路由器端口转发（需要配置）

在你的路由器上设置端口转发，让外网能访问本地服务器：

操作步骤：

  1. 登录路由器管理页面（通常是192.168.1.1）
  2. 找到"端口转发"或"虚拟服务器"设置
  3. 添加新规则：
    • 内部IP：192.168.1.152
    • 内部端口：8899
    • 外部端口：8899（或其他）
    • 协议：TCP
  4. 保存设置，重启路由器
  5. 获取公网IP（百度搜"我的IP"）
  6. 访问：http://[公网IP]:8899/
// 访问地址示例：
http://123.123.123.123:8899/

⚠️ 方案三：使用内网穿透工具（技术性较强）

需要安装和配置工具：

可选工具：

  1. ngrok - 简单但需要注册
    # 需要注册账号获取token
    ngrok http 8899
  2. frp - 免费开源，需要服务器
    # 需要自己搭建frp服务器
    frpc -c frpc.ini
  3. ZeroTier - 创建虚拟局域网
    # 安装客户端，加入网络
    zerotier-cli join [网络ID]
